x402-casper-api

Pay-per-query for Casper blockchain data. No API keys. No accounts. No subscriptions.

Send CSPR → get data. One transfer. One query. $0.05.

How it works

1. Send CSPR → 2. Get deploy hash → 3. Query with X-402-Payment header

Server calls info_get_deploy on-chain to verify the transfer reached the receiver. If valid, data is returned. No mock verification. No string parsing. Real RPC.

Payment flow

Send CSPR to the receiver address configured in .env. Take the deploy hash and pass it as the X-402-Payment header value. The server fetches the deploy via info_get_deploy, checks execution success, confirms the transfer target matches the configured receiver, and returns the data.
